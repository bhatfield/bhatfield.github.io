Dashboard 2.1.1 Release Notes

The 2.1.1 release of Appcelerator Dashboard contains several bug fixes, workflow improvements for adding team members and inviting organization members, support for plain-text email templates, the ability to enable or disable iOS push notification certificates.

  • New Features and Improvements

    • Improved workflow for adding app team members

    • Improved workflow for inviting organization members

    • Cloud and push notification logs improvements

    • Support for creating plain text email templates

    • Ability to enable/disable iOS push notification certificates

  • Bug Fixes

New Features and Improvements

Improved workflow for adding app team members

When creating a new application, a new form at right lets you immediately add users to the application team.

Similarly, when creating a new user you can quickly add them to application teams.

Improved workflow for inviting organization members

When creating a new organization, a new form at right lets you immediately invite members to join the organization.

Cloud and push notification logs improvements

Log tables for pre-built Cloud API calls and push notifications now provide next/previous pagination buttons, instead of page numbers.

Also, you can now filter Cloud API logs by date, and push notification logs by notification channel name or creation date.

Support for creating plain text email templates

Email templates you create in Dashboard can now have plain text as well as HTML-formatted bodies. When sending an email based on a template with both plain text and HTML body types, you can specify whether you want to send the email using HTML body, plain text or both (multipart).

Ability to enable/disable iOS push notification certificates

Imported iOS push certificates can now be enabled or disabled.
