Dashboard 3.0.2 Release Notes

The 3.0.2 release of Appcelerator Dashboard contains minor improvements and several bug fixes.

  • Improvements

  • Bug Fixes

Improvements

To streamline the activation process for new organization members, the user activation dialog and end-user license agreement (EULA) dialog have been combined.

Bug Fixes

Below is a select list of bugs fixed in the Dashboard 3.0.2 release.

Summary

Fixed "Export Data" button on the Analytics Events page.

Fixed issue saving a Status object after editing it.

Fixed issue with page load after deleting ACS application data.

Fixed "more info" links that are provided to Appcelerator Studio.

The API menu now updates immediately after adding or removing an API.

Fixed version number displayed in the Download Center for the Appcelerator Crypto Module.

Fixed column alignment on Service API Apps page.

Fixed an issue filtering ACS log items whose time-stamp equals a specific date and time.

Links to enable Test and Performance services now open the correct instructions.

Fixed alignment of the Owner and Place Add buttons when modifying ACS objects.
