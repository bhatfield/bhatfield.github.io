Dashboard 2.3.0 Release Notes

The 2.3.0 release of Appcelerator Dashboard introduces a new top-level menu that lets you quickly access your ACS and Node.ACS services, a simplified user provisioning workflow and other improvements, as well as many bug fixes.

New Features and Changes

APIs Menu

Dashboard 2.3.0 introduces a new top-level APIs menu that lists all ACS and Node.ACS APIs in your organization. ACS services are indicated by "data" in the menu label while Node.ACS services are indicated by "services" in the label, as shown below.

To view a list of APIs in the currently selected organization, select All APIs from the APIs menu. The APIs page lists the total number of APIs in the organization, their type and the date they were created.

Each application in the Apps menu has a corresponding item in the APIs menu with the same name. For instance, if you have an application named "ChatApp", select APIs > ChatApp to manage that application's service.

API Management Page

Selecting an API from the APIs menu, or from the list of all APIs (shown above), opens a page where you can view and manage the API, as shown below. Previously, this functionality was accessed by clicking the "Cloud" menu while viewing an application. The Cloud menu has been removed from Dashboard (see Removal of Cloud Menu Item). For details, see Managing Arrow DB Applications.

Removal of the Cloud Menu Item

In previous Dashboard releases, to view or manage an application's ACS or Node.ACS service you would first select the application from the Apps menu, and then select Cloud from the secondary navigation. In Dashboard 2.3.0, the Cloud menu item has been removed. Instead, you can now simply select the API you want to manage from the APIs menu in the top-level navigation, as discussed in Managing Arrow DB Applications.

Other Improvements

  • Chat messages are now displayed in the same context as the selected chat group, rather than in a separate view.

  • Improved error messages for filter operations on ACS data.

  • Improved icons for the table/tile view toggle button on the list of Apps.
