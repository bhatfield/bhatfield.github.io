Importing a Sample

You can import any of these samples from the Studio start page or from the Samples view.

To import a sample from the start page:

  1. If the Dashboard is not open, in the menu bar, select Help > Studio > Show Studio Dashboard.

  2. Click on the Develop tab on the dashboard.

  3. Find the sample you want to import and click the Import button.

  4. Follow the wizard prompts to finish importing the project.

To import a sample from the Samples view:

  1. If the Samples view is not open, in the menu bar, select Window > Show View > Samples.

  2. Right-click on a project and click Import sample as project..., or select a project and click the import button ( ).

  3. Follow the wizard prompts to finish importing the project.

RSS Reader

With this sample, you can pull live RSS feeds from the internet, list them along with thumbnails, then drill down to the article itself.

Topics Covered

  • Remote Data acccess via Titanium.Network.HTTPClient

  • Modular Javascript with CommonJS

  • Multiple window app using a single execution context

  • Loose coupling of UI and data code using application level events via Ti.App

  • Native UI features

    • Android menus

    • iOS navigation bar buttons

    • iOS Navigation Controller via Titanium.UI.iPhone.NavigationGroup

  • Webviews for displaying web content

  • Cross-platform design
