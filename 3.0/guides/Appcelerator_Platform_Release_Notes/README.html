Appcelerator Platform Release Notes

  • Appcelerator Platform Summer 2014 Release Notes (17 July 2014) &mdash; This release contains major updates to Appcelerator Dashboard, Appcelerator Cloud Services, Titanium SDK and Appcelerator Studio, introduced the Appcelerator Platform Services SDKs for iOS and Android platforms. It included 650 improvements and new features, and addressed 200 customer requests.

  • Appcelerator Platform 2.1.1 Release Notes &mdash; The 2.1.1 release of Appcelerator Platform includes the Appcelerator Dashboard 2.1.1 release, Titanium SDK 3.4.1 and Appcelerator Studio 3.4.1 releases, as well an update to the Appcelerator Test service.

  • Appcelerator Platform 2.1.0 Release Notes (04 November 2014) &mdash; The 2.1.0 release of Appcelerator Platform includes several new features and improvements, updates to the Appcelerator Cloud Services (1.1.6) and an update to the Appcelerator Test service.

  • Appcelerator Platform 2.0.1 Release Notes (16 October 2014) &mdash; The 2.0.1 release of Appcelerator Platform includes updates to Appcelerator Dashboard and Insights (2.0.1), Appcelerator Cloud Services (1.1.5), Titanium SDK and Appcelerator Studio (3.4.0), and Appcelerator Test (7198.215).
