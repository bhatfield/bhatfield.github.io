Appcelerator Platform 2.1.1 Release Notes

The 2.1.1 release of Appcelerator Platform includes the Appcelerator Dashboard 2.1.1 release, Titanium SDK 3.4.1 and Appcelerator Studio 3.4.1 releases, as well an update to the Appcelerator Test service.
