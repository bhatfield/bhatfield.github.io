Dashboard 2.0.1 Release Notes

The 2.0.1 release of Appcelerator Dashboard contains many bug fixes, as well as several new features and improvements.

  • UI and Navigation

  • Usability

  • Push Notifications

  • Other

  • Known Issues

  • Bug Fixes

UI and Navigation

  • Added ability to collapse the left navigation bar, and improved visibility of navigation selection.

Usability

  • The Email Template form now includes a sample email HTML template.

  • Added ability to search custom events by name.

  • A new modal feedback form has replaced the feedback input field.

  • Currently running applications are listed first in the Node.ACS analytics overview screen.

Push Notifications

The Send Push Notification form now includes a Category field for working with interactive notifications available in iOS 8, and a Content-Available field to support Newsstand apps and background content downloads. For more information, see Interactive Push Notifications and Silent Push Notifications in Sending and Scheduling Push Notifications.

Other

  • Added demo data for Events and Node.ACS API calls.

Known Issues

  • The number of sessions displayed on the pins plotted on the Sessions > Map view does not match the numbers displayed in the corresponding data table. This issue is specific to native Android applications using the Platform Services SDK.

  • The average session length displayed for the Sessions metric is longer than expected due to sessions not terminated properly. This is will be fixed in the next release.

  • For native Android applications, the geolocation of map pins displayed for Installs and Sessions may be located at (0,0).
