Dashboard 3.0.1 Release Notes

The 3.0.1 release of Appcelerator Dashboard contains the following changes and bug fixes.

  • Improvements and Changes

  • Bug Fixes

Improvements and Changes

  • Dashboard now indicates whether or not a Node.ACS service is running.

  • For organizations running Dashboard in virtual private cloud (VPC) environment, API names are now appended with the name of the VPC environment.

  • Appcelerator LDAP Module was added to the Download Center.

  • Implemented error pages for server error codes 502 and greater.

Bug Fixes

Below is a select list of bugs fixed in the Dashboard 3.0.1 release.

Summary

Fixed issue paging through ACS GeoFences objects in Dashboard.

Long URLs are now properly wrapped in Node.ACS logs.

Fixed an issue where adding a custom field to an ACS user profile changed the user's password.

Fixed an issue with Demo Data mode where the number of active sessions goes to 0.

Fixed an issue navigating the menu of organizations with the keyboard.
