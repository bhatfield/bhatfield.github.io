Dashboard 2.2.0 Release Notes

The Appcelerator Dashboard 2.2.0 release contains changes and improvements for viewing and managing applications, and several bug fixes.

  • New Features and Improvements

    • Application UI changes

    • Cloud UI changes

    • General UI changes

  • Bug Fixes

New Features and Improvements

Application UI changes

The UI for how you view and manage applications and application teams in your organization has been improved:

  • The list of applications has been moved from Manage Orgs to its own page. Select All Apps from the Apps menu to view the list of apps.

  • When viewing details for an application, a new Team menu lets you quickly view and manage an application's team members. (This feature was previously accessible from the Manage Orgs > Manage Apps page.)

  • You can now view your applications in a grid layout, as well as the standard tabular layout.

  • You can now search for an application by its GUID, as well as application name.

Cloud UI changes

  • A new Documentation page provides links to ACS documentation and community resources.

  • Push notification settings have been moved to the Cloud > Push Notifications page. They were previously located in the Cloud > Configuration page.

General UI changes

The 2.2.0 release includes general UI enhancements that make navigation easier and better indicate your place in the UI.

  • Your current menu selection within the Dashboard UI is more clearly indicated.

  • The name of the currently selected application is now clearly displayed in the left-hand navigation menu, instead of the top-level navigation bar.
