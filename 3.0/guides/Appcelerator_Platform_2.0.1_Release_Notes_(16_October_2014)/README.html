Appcelerator Platform 2.0.1 Release Notes (16 October 2014)

The 2.0.1 release of Appcelerator Platform includes updates to Appcelerator Dashboard and Insights (2.0.1), Appcelerator Cloud Services (1.1.5), Titanium SDK and Appcelerator Studio (3.4.0), and Appcelerator Test (7198.215).
