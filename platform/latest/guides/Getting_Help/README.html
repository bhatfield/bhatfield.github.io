Stuck? We're here to help!

Overview

We all get by with a little help from our friends. Here are a few ways to get help using and mastering Titanium.

Reference Documentation

In addition to these guides, more information and examples are available in the API Reference.

Q&A

We have a community-driven question and answer board that is primarily supported by your fellow Titanium developers. Chances are good that someone has already seen an issue similar to yours, so this is the best place to start searching when you have a question. If no one has asked your question, go ahead and fire off your own.

Community Resources

As the Titanium community has grown, so too have the resources on Titanium available out on the web. Every day we see more blogs, tutorials, and snippets posted about Titanium, which is great. Firing up your favorite search engine will yield good results if you can't immediately find the answer you're looking for in the Docs or Q&A.

Pro Support

For SLA-driven support directly from the Appcelerator team, you can upgrade to a Team or Enterprise subscription. An upgraded subscription gives you access to a premium support help desk staffed by Appcelerator engineers. You're also directly supporting the great open source projects that we make available to the world free of charge - how cool are you!
